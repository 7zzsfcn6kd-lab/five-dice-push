Rules
  • Roll five dice up to three times.
  • Twos are wild and count toward the strongest multiple.
  • After each roll, hold any dice and reroll the rest.
  • The challenger gets only as many rolls as the target used.
  • More matching dice wins. Ties go to higher face value, then fewer rolls.
  • An exact match is a push: no point, same starter leads again.
